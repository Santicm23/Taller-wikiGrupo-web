Descripción del Proyecto

Primeramente el taller tiene como objetivo principal la creación de una "wiki" para nuestro equipo. En esta wiki, se documentarán detalladamente los aspectos más relevantes del proyecto final en desarrollo.

Servirá como un recurso valioso para los miembros del equipo, permitiéndoles acceder rápidamente a la información relevante y mantenerse al tanto de los avances del proyecto.

El proyecto podría tratarse de una plataforma o aplicación de votación y gestión de listas de reproducción de música, donde los usuarios pueden registrarse, autenticarse y votar por sus canciones favoritas. Los usuarios votantes pueden crear y ver listas por género musical, buscar canciones y votar o desvotar por las listas existentes. Los usuarios administradores, por otro lado, pueden autenticarse y gestionar los géneros musicales y canciones mediante operaciones CRUD (Crear, Leer, Actualizar y Borrar).

Requerimientos Funcionales

  • Historias de Usuario para el Usuario Votante:
  • Creación de Cuenta
  • Activación de Cuenta
  • Autenticación de Usuario
  • Ver Listas por Género Musical Creadas
  • Buscar Canción
  • Votar/Desvotar por una Lista
  • Historias de Usuario para el Usuario Administrador:
  • Autenticación de Usuario
  • CRUD Géneros Musicales
  • CRUD Canciones

Requerimientos No Funcionales

  • Seguridad de Datos
  • Interfaz de Usuario Intuitiva
  • Tiempo de Respuesta Rápido
  • Implementación Robusta
